## Usage

Mainly run `main_running_mode.py`.

In this file, experimental parameters and port numbers can be defined.

## Modules Description

- **Motor fin**  
  Controls the Dynamixel motor fin.

- **Motor flow**  
  Controls the flow.

- **Motor step**  
  Controls the 3D position platform for both leader and follower.

- **Sensor single**  
  Mounts the F/T sensor and acquires 6-axis data.  
  To measure both sensors, they must be mounted separately with different port numbers.

- **Signal pulse**  
  Controls the generation of pulse signals.
